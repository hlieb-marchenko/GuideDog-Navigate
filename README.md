# 🦮 GuideDog Navigate

An assistive navigation application designed to help visually impaired users navigate environments safely and independently using real-time guidance, routing, and accessibility-focused interaction design.

---

## ✨ Overview

GuideDog Navigate is an assistive navigation system inspired by the behavior of guide dogs. It aims to provide users with a reliable, accessible, and intuitive way to move through physical spaces using digital guidance.

The system focuses on:

- Safe navigation assistance
- Clear step-by-step instructions
- Accessibility-first interaction design
- Reduced reliance on visual interfaces

---

## 🚀 Features

- 📍 Real-time GPS tracking and location updates
- 🗺️ Route planning and navigation support
- 🔊 Voice-guided navigation instructions
- 🧭 Step-by-step directional assistance
- ♿ Fully accessibility-oriented UI design
- 📱 Mobile-friendly responsive interface

---

## 🧠 How It Works

The application acts as a digital "guide companion":

1. User selects a destination
2. System calculates optimal route
3. App continuously tracks user location
4. Voice system provides real-time instructions
5. Navigation updates dynamically based on movement

## ⚙️ Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/hlieb-marchenko/GuideDog-Navigate.git
cd GuideDog-Navigate
```

### 2. Install dependencies

**Frontend:**

```bash
npm install
```

### 3. Run the application

**Frontend:**

```bash
npm start
```
